# Python Course

A comprehensive Python learning course covering fundamental programming concepts.

## Course Modules

| Module | Topic | Description |
|--------|-------|-------------|
| 01_Hello.py | Hello World | Introduction to Python and basic output |
| 02_Variables.py | Variables | Understanding variables and data storage |
| 03_String.py | Strings | String manipulation and operations |
| 04_Integer.py | Integers | Integer operations and properties |
| 05_Float.py | Floating Point | Float numbers and decimal operations |
| 06_Function.py | Functions | Defining and using functions |
| 07.0_Conditionals.py | Conditionals | If/else statements and logical operators |
| 07.1_Parity.py | Parity Check | Working with even/odd number detection |
| 07.2_Home.py | Home Project | Practical conditional exercise |
| 08_Loops.py | Loops | For and while loops |
| 09_List.py | Lists | List data structures and operations |
| 10_Dictionary.py | Dictionaries | Dictionary data structures and operations |

## Getting Started

1. Start with module 01_Hello.py and progress sequentially
2. Each module builds on concepts from previous modules
3. Run each script to see the output and understand the concepts

## Prerequisites

- Python 3.6 or higher installed on your system
- A text editor or IDE (VS Code recommended)

## Running the Scripts

To run any script, open a terminal and execute:

```bash
python <module_name>.py
```

Example:
```bash
python 01_Hello.py
```

## Learning Path

The course follows a progressive learning path:
- **Basics** (01-05): Output, variables, and data types
- **Control Flow** (06-07): Functions and conditionals
- **Iteration** (08): Loops and repetition
- **Data Structures** (09-10): Lists and dictionaries

## Tips for Success

- Type out the code instead of copying and pasting
- Experiment with modifying the examples
- Try solving problems before looking at solutions
- Practice regularly to reinforce concepts
